## HEMTT (Installation and Use)
HEMTT is the modding framework we use for the Auxiliary Mod.  
Primarily the functions allow for easier testing and easy building of the mod.  

**Inital Installation** (Windows 10/11)
1. Open any CMD(Powershell/Command Prompt).
2. Run 'winget install hemtt'.
3. Follow the installation prompts and agree to the terms.
4. Exit your CMD.
5. Github Desktop, Visual Studio Code must be restarted before use.

**Mod Testing**
1. Open Visual Studio Code from Github Desktop. (This ensures the correct path.)
2. Open Visual Studio Code terminal.
3. Run 'hemtt dev'.
-? First time? Follow these next steps as well. ?-  
(1.) Open the mod directory.  
(2.) Open .hemttout.  
(3.) Drag the 'Dev' folder into the Arma 3 Launcher.  
4. Launch Arma 3. (Make sure to unload the actual aux mod and automatic warning supression, load advanced developer tools for config work.)

There are more commands available in hemtt. 
Run 'hemtt help' for the full list, and or visit their wiki (https://hemtt.dev/index.html)

## AUXILIARY MOD UPDATE (Instructions/Guide)
You must complete HEMTT inital installation before doing this.

1. Confirm that the aux mod is ready for update. (Speak with the Config Team.)  
2. Open Visual Studio Code from Github Desktop.  
3. Open Visual Studio Code terminal.  
4. Run 'hemtt release'.  
5. Open the mod directory. 
6. Open .hemttout.  
7. Use Arma 3 Publisher Tool to upload the 'release' folder.  
8. Follow standard update procedures.
